#### 🚧 安装 cnpm、yarn

```bash
# node 版本
node -v 
v14.17.5
```

- 复制代码(桌面 cmd 运行) `npm install -g cnpm --registry=https://registry.npm.taobao.org`
- 复制代码(桌面 cmd 运行) `npm install -g yarn`

```bash
# 克隆项目
git clone https://github.com/baizunxian/zero_autotest_front

# 进入项目
cd zero_autotest_front

# 安装依赖
cnpm install 
# 或者
yarn

# 运行项目
cnpm run dev
# 或者 
yarn dev

# 打包发布
cnpm run build
# 或者 
yarn build
```
